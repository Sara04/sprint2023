What?

Holy sh*t! We're in a paradigm shift.

In case you haven't noticed, machine learning and scalable computing have - in less than a decade - demonstrated how a rich set of information can be extracted from complex datasets, with consequences in most scientific and industrial fields.

Here, we are organizing a coding sprint to gather established data-scientists, who specialize in high dimensional time series, to work on a common issue: integrating, within a coherent Python ecosystem, classical signal processing and popular machine learning techniques.

We focus on increasing the inter-operability of open-source packages developed for particularly complex yet manipulable data: neuronal recordings (e.g. animal electrophysiology, electro-corticography and magneto-/electro-encephalography (M/EEG)). And we impose a simple constraint: functions and packages must follow the scikit-learn's API.

This framework will allow users to quickly build effective pipelines to fit, make predictions from and score complex models on raw or transformed time series, within a principled workflow:

{% highlight python %} # Example of a simple decoding pipeline using spike recordings. estimator = make_pipeline( UnsupervisedSpatialFilter(PCA(nc_components=5)), SpikeDetection(threshold='auto'), SearchLight(GLM(link_function='poisson'), axis='time'), ) {% endhighlight %}

{% highlight python %} # Example of a simple decoding pipeline using classic brain-computer interface algorithm. estimator = make_pipeline( CSP(n_components=10), InverseOperator(info), SearchLight(Ridge(), axis='space'), ) {% endhighlight %}

{% highlight python %} # Example of an advanced decoding pipeline using MEG and Riemannian geometry. estimator = make_pipeline( TimeFrequencyTransform(frequencies=np.arange(1., 100.), sfreq=1000.), SearchLight( make_pipeline( Covariances(regularization='oas'), TangentSpace(), SVM(kernel='rbf')), axis='time'), ) {% endhighlight %}

{% highlight python %} estimator.transform(X, Y) # transform neural time series. estimator.fit(X, Y) # transform & fit neural time series with machine learning algorithm. estimator.predict(X) # predict features of an unknown time series. estimator.score(X) # estimate quality of a model fit. {% endhighlight %}

This effort will directly i) tighten the neuroscience and machine learning communities ii) increase the modularity and robustness of Python eco-system, and therefore iii) facilitate scientists' daily work by producing robust and useable cutting-edge tools.
